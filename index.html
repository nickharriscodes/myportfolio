About Me

Hi there! I’m a web developer, musician, cyclist, and plant dad based in Chicago, Illinois. I graduated with from Loyola University Chicago in 2013 with a BA in English, and since then I have worked as a copywriter, retail shift manager, music teacher, among other things, all the while writing music and recording with several different bands.

I started learning about web development a little over a year ago when I received a Google Challenge Scholarship to complete a FrontEnd-focused Nanodegree with Udacity. Since completing my studies with Udacity, I have continued feeding my curiosity and passion for learning by self-teaching using the boundless resources of the internet. Key Projects


Matching Game

This application is a fun card-matching memory game that I created while working on my Udacity Nanodegree. I built out all of the dynamic functionality of this game — the “flipping” cards, matching logic, scoring, timer — using vanilla JavaScript, and it helped me solidify some fundamental concepts like object-oriented programming, version control, scope, and more.

Album Vortex

A wacky project created as a way to experiment with CSS animation and a review of some basic JS techniques like event handling, this application consists of a spinning vortex of some of my favorite albums of all time. Press the spacebar to make it stop, or let it keep spinning and click at random to get more info on the album you happen to land on.

Book Search

An app I built as a learning-project in order to work with the Google Books API. The query entered in the search box returns a list of books and corresponding information which has been filtered from a JSON object using jQuery. There are various user-friendly ways to clear the results and run a new search. This app was also subject to a code review from an industry professional, and revised and improved in response to that review.
