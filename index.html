About Me

Hi there! I’m a web developer, musician, cyclist, and plant dad based in Chicago, Illinois. I graduated from Loyola University Chicago in 2013 with a BA in English, and since then I have worked as a copywriter, retail shift manager, music teacher, among other things, all the while writing music and recording with several different bands.

I started learning about web development a little over a year ago when I received a Google Challenge Scholarship to complete a FrontEnd-focused Nanodegree with Udacity. Since completing my studies with Udacity, I have continued feeding my curiosity and passion for learning by self-teaching using the boundless resources of the Internet. Below you’ll find a few links to live examples of my best work thus far.

Key Projects


Matching Game

This application is a fun card-matching memory game that I created while working on my Udacity Nanodegree. I built out all of the dynamic functionality of this game — the “flipping” cards, matching logic, scoring, timer — using vanilla JavaScript, and it helped me solidify some fundamental concepts like object-oriented programming, version control, scope, and more.

Band Website

This is the official website for the Chicago-based alternative rock band Young Man in a Hurry, which will release its debut album in 2019. A mostly informational website built using bootstrap for enhanced styling, this website is used as a promotional tool by the band. I collaborated with Hartford Designs and the client and responded to feedback to create this page.

Book Search

An app I built as a learning-project in order to work with the Google Books API. The query entered in the search box returns a list of books and corresponding information which has been filtered from a JSON object using jQuery. There are various user-friendly ways to clear the results and run a new search. This app was also subject to a code review from an industry professional, and revised and improved in response to that review.
